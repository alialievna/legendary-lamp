# 📊 Analytics Workflow

Автоматический аналитический сервис для данных маркетплейса.  
Никакой ручной работы после загрузки файлов. Никаких внешних AI API.

## Запуск локально

```bash
# 1. Клонировать / скачать репозиторий
cd analytics-service

# 2. Установить зависимости
pip install -r requirements.txt

# 3. (Опционально) Для корректного отображения кириллицы в PDF
#    скачать шрифт DejaVu Sans:
python download_fonts.py

# 4. Запустить
streamlit run app.py
```

Сервис откроется на http://localhost:8501

## Форматы входных файлов

### sales.csv
```
дата,sku,название_товара,категория,продано_штук,выручка,сессии
2025-09-01,SKU-001,Товар 1,Электроника,19,17708,449
```

### returns.csv
```
дата,sku,количество_возвратов,причина_возврата
2025-09-01,SKU-001,4,Возврат клиента
```

### stock.csv
```
sku,остаток,склад
SKU-001,218,Склад-СПБ
```

### campaign_info.txt
Произвольный текст — описание маркетинговой акции (первая строка используется в сводке).

## Логика алертов

| Уровень | Условие |
|---------|---------|
| 🔴 Критично | % возвратов > 20% |
| 🟡 Внимание | % возвратов 15–20% |
| 🔴 Критично | Остаток < 50 шт. |
| 🟡 Внимание | Остаток 50–100 шт. |
| 🔴 Критично | Продажи = 0 за последние 7 дней |
| 🟡 Внимание | Снижение продаж > 40% (последние 5 vs первые 5 дней) |
| 🟡 Внимание | Конверсия < 2% |

## Деплой на Streamlit Community Cloud (бесплатно)

1. Залить проект на GitHub (публичный репозиторий)
2. Зайти на https://share.streamlit.io
3. Войти через GitHub
4. **New app** → выбрать репозиторий → ветка `main` → файл `app.py`
5. **Deploy** → через 2–3 минуты сервис доступен по постоянной ссылке

> `packages.txt` автоматически установит шрифты DejaVu для корректного PDF с кириллицей.  
> Никаких секретов и API-ключей не требуется.

Ссылка на сервис будет вида: `https://your-app-name.streamlit.app`

## Структура проекта

```
analytics-service/
├── app.py                  # главный файл Streamlit
├── requirements.txt        # Python-зависимости
├── packages.txt            # системные пакеты (шрифты для PDF)
├── download_fonts.py       # хелпер для локальной разработки
├── .streamlit/
│   └── config.toml         # тема и настройки
├── utils/
│   ├── parser.py           # парсинг и валидация CSV
│   ├── metrics.py          # расчёт KPI
│   ├── alerts.py           # логика алертов
│   ├── summary.py          # генерация текстовой сводки
│   └── export.py           # генерация PDF и CSV отчётов
└── README.md
```

## Стек

- **Frontend + Backend**: Streamlit
- **Анализ данных**: pandas
- **Графики**: plotly (интерактивные)
- **PDF-экспорт**: reportlab
- **AI/ML**: не используется
